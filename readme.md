### Контрольные вопросы 

1. Что такое массивы в PHP?

Массив в PHP — это упорядоченная структура данных, которая хранит набор значений (элементов). Каждый элемент состоит из ключа (индекса) и значения. Массивы могут быть индексированными (ключи — целые числа) и ассоциативными (ключи — строки). В PHP массивы реализованы как хеш-таблицы, что обеспечивает быстрый доступ к элементам по ключу.

2. Каким образом можно создать массив в PHP?

* Существует два основных способа создания массива:

* С помощью языковой конструкции `array()`:
`$arr = array(1, 2, 3)`;

* С помощью короткого синтаксиса `[]` (начиная с PHP 5.4):
`$arr = [1, 2, 3];`
Для создания ассоциативного массива используется синтаксис `ключ => значение`:
`$user = ['name' => 'John', 'age' => 30]`;

3. Для чего используется цикл `foreach`?

Цикл `foreach` предназначен для перебора всех элементов массива (или объекта). Он последовательно извлекает ключ и значение каждого элемента, что позволяет удобно обрабатывать данные без необходимости вручную управлять счётчиком или условиями выхода.
